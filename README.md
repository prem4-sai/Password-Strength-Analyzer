# 🔐 Password Strength Analyzer


## 📌 Project Description


The Password Strength Analyzer is a Python-based application that evaluates the strength of a user's password using multiple security criteria. It checks password length, uppercase letters, lowercase letters, numbers, special characters, common weak passwords, repeated characters, and sequential patterns. Based on these checks, it classifies the password as **Weak**, **Medium**, or **Strong** and provides suggestions to help users create stronger and more secure passwords.

---

## 🎯 Objectives

## ▶️ How to Run

1. Install Python 3.
2. Open the project in Visual Studio Code.
3. Open the Terminal.
4. Run the following command:

```bash
python password_strength_analyzer.py
```

5. Enter a password.
6. View the password strength and suggestions.
